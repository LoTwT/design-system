# Focus Ring

V0 ships two focus utilities:

```css
focus-ring
focus-ring-inset
```

  <button class="touch-target rounded-control border px-4 focus-ring-inset" style="border-color: var(--border-default); background: var(--surface-elevated); color: var(--text-primary);">
